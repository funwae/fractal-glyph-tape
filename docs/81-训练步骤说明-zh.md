# 训练步骤说明（FGT 模型）

## 1. 数据准备

1. 运行 ingest：

   - 得到 `data/phrases.jsonl`。

2. 构建 FGT tape：

   - embed → cluster → glyph → tape。

## 2. 模型训练流程

1. 准备训练样本：

   - 输入：glyph 编码的文本（或混合文本）。

   - 标签：原始文本、任务标签等。

2. 训练目标：

   - 重构损失（reconstruction）。

   - glyph 预测损失（glyph prediction）。

   - 下游任务损失（可选）。

3. 训练脚本：

   - `python scripts/train_with_fgt.py --config configs/train.yaml`

## 3. 检查点

- 保存：

  - 模型权重。

  - Tokenizer / FGT 配置。

  - 实验日志和评估结果。
