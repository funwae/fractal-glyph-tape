# glyphd.com Integration

We describe how to present FGT on glyphd.com.

## 1. Landing page

Content:

- Plain-English explanation.

- High-level diagram of pipeline.

- Embedded fractal map visualization (read-only demo).

- Links to:

  - GitHub repo.

  - Research results.

## 2. Interactive demo

Options:

- "Paste text" demo:

  - Show glyph-coded version.

  - Show phrase families activated.

- "Explore map" demo:

  - Precomputed tape with limited dataset.

## 3. APIs

Expose limited FGT APIs behind glyphd.com:

- `/api/fgt/encode`

- `/api/fgt/inspect-glyph`

- etc., rate-limited.

## 4. Positioning

Messaging:

- FGT as a **research infrastructure**.

- Glyphd as the "lab" building new substrate for cross-language AI.
