# 数据格式与约定

## 1. 短语文件

- `data/phrases.jsonl`

- 字段：

  - `phrase_id` (int)

  - `text` (str)

  - `lang` (str)

  - `doc_id` (str)

## 2. 嵌入分片

- `embeddings/shard_XXX.npy`

- meta: `embeddings/meta.json`

## 3. 聚类结果

- `clusters/labels.npy` – phrase → cluster。

- `clusters/centroids.npy` – 聚类中心。

- `clusters/metadata.jsonl` – 聚类元信息。

## 4. tape 存储

- `tape/vX/clusters_table.npy`

- `tape/vX/glyph_to_cluster.sqlite`

- `tape/vX/address_to_clusters.sqlite`

所有路径与字段名保持稳定，便于脚本和测试。
