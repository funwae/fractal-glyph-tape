# 评估流程（FGT）

## 1. 压缩评估

- 步骤：

  - 对 eval 语料进行 FGT 编码。

  - 计算原始字节数 vs FGT 字节数。

  - 重构为文本，计算 BLEU / BERTScore。

## 2. 上下文效率

- 固定 token 限制 L。

- 分别使用：

  - 原始文本。

  - FGT 编码文本。

- 比较 QA / 摘要任务的表现。

## 3. 多语言检索

- 构造跨语言检索任务。

- 比较：

  - 普通 embedding 检索。

  - FGT + glyph ID 检索。

## 4. 人工评估

- 语义保持 / 流畅度打分。

- 聚类一致性评估。

详细指标见 `60-eval-metrics-overview.md`。
